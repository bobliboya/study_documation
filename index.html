HOW TO USE

How to Get Access to IFSP?

IFSP is available on all operation systems. Because of our HTML5 responsive webpage, you can visit IFSP on any devices. Now you can use any popluar web browser to get access to IFSP. The link is https://bobliboya.github.io/docute-starter. If the connection speed is not fast, you may try to use a proxy or visit https://bobliboya.gitee.io/docute-starter. (This link can improve the speed in Mainland China.)

Can i develop these documents?

The answer is YES. IFSP will be an open-source project. Please contact the administrator of IFSP.
